## ✨url에 www.google.com을 입력하면 일어나는 일✨

**search query인지 URL인지 판단한다.**

**DNS에서 IP 주소를 추출한다.**
> (1) 브라우저 캐시를 확인한다.
> (2) OS 캐시를 확인한다.
> (3) 라우터 캐시를 확인한다.
> (4) ISP 캐시를 확인한다.
> (5) 브라우저가 DNS 쿼리를 사용하여 DNS를 탐색환다.

**서버와 TCP 연결을 시도한다.**

**웹 서버에 HTTP 요청을 보낸다.**

**서버는 요청을 처리하고 response를 생성한다**

**브라우저에 HTTP 응답을 보낸다**

**브라우저는 HTML 컨텐츠를 렌더링하여 표시한다**
