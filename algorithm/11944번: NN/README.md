## 문제

[문제 링크](https://boj.kr/11944)


<p>문제는 매우 간단하다. N을 N번 출력하는 프로그램을 작성하여라. 다만, 답이 길어지는 경우 답의 앞 M자리만 출력한다.</p>




## 입력


<p>첫 번째 줄에는 N, M이 주어진다. (1 ≤ N, M ≤ 2016)</p>




## 출력


<p>N을 N번 출력한다. 만약 답이 길어지면 답의 앞 M자리를 출력한다.</p>
